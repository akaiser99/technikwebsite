# Technikhilfe Stuttgart – Website

## Online über GitHub Pages stellen
1. Auf GitHub ein neues Repository erstellen, z. B. `technikhilfe-stuttgart`.
2. Alle Dateien aus diesem Ordner hochladen.
3. Im Repository auf **Settings → Pages** gehen.
4. Unter **Build and deployment** bei **Source** `Deploy from a branch` auswählen.
5. Branch `main` und Ordner `/(root)` auswählen und speichern.
6. Nach kurzer Zeit zeigt GitHub dort deine Website-Adresse an.

## Vor dem Veröffentlichen ändern
- In `index.html` die E-Mail-Adresse ändern, falls du eine neue Geschäftsadresse anlegst.
- In `impressum.html` alle Angaben in eckigen Klammern durch echte Pflichtangaben ersetzen.
- In `datenschutz.html` die gleiche verantwortliche Person/Adresse ergänzen.
- Prüfe, ob du die Tätigkeit gewerblich angemeldet hast und welche Angaben für dich gelten.

Die Seite arbeitet ohne externe Bilder, Tracking und ohne Datenbank. Das Anfrageformular öffnet nur das E-Mail-Programm des Besuchers.
